To run the application Apache Hadoop, Apache Hive, Apache HBase and REST in HBase master node has to be
up and running. All the configuration and setting details of each technology can be found in following links:

http://wiki.apache.org/hadoop/
https://cwiki.apache.org/confluence/display/Hive/Home
http://hbase.apache.org/

Following libraries were used for the application development:

1. Java libraries version 1.6
2. Core libraries of hadoop-1.0.4
3. Core libraries of hbase-0.94.7
4. Core libraries of hive-0.10.0

All these libraries can be found on the links above.
